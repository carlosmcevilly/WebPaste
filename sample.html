Things to try:

  • Tap Check to see the list of pasteboard data types that are currently in the pasteboard.
  • Copy some text from this web view, then try Check again.
  • Copy "Pasteboard Data Types:" from the text view, then try Check again.
  • Tap Make to have the app programmatically put HTML content on the pasteboard, then try Check.
  • Copy the Google link above, then go to an HTML-capable app (like the Mail composition view) and do a paste. Try the same thing with a plain text app (like Notes).
